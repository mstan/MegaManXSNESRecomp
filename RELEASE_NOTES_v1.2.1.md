# Mega Man X (Recompiled) v1.2.1 — shared ImGui launcher

This release moves the pre-boot UI to the current shared Dear ImGui `recomp-ui`
launcher. Its DPI-independent layout fits the launcher controls on smaller
1366×768 displays and keeps the Play action reachable, addressing issue #9.

- Shared SNES launcher profile, controller/hotkey editor, ROM verification, and
  display/audio settings.
- Current launcher fonts, controller art, and game box art bundled in the
  Windows package.
- Existing gameplay, widescreen, and native-analysis changes from main are
  otherwise unchanged.
